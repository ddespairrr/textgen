Скрипт make_model.py принимает на вход в качестве аргумента путь до файла с текстом, на основе которого требуется
построить биграммную языковую модель. Затем он сохраняет модель в файле model.json
Скрипт main.py принимает на вход в качестве аргумента путь до файла с моделью и количество предложений,
которое будет сгенерированно
